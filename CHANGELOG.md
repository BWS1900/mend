## [0.1.0] - 2026-XX-XX

### Added
- Initial release.
- CommonMark + GFM (tables, task lists, strikethrough, autolinks).
- Footnotes, definition lists, smart typography.
- Syntax highlighting via chroma.
- Themes: `github`, `github-dark`, `monokai`, `nord`, `none`.
- Watch mode (`-w`) and HTTP live reload (`-serve`).
- Stdin/stdout, file input/output, fragment output.
- Single static binary, cross-compiled for linux/darwin/windows on amd64/arm64.
- Homebrew, Scoop, `go install`, and curl-based install.
